# 汎用人格搭載通貨「I'm COIN」

## ✏️コンセプト

人格AIが搭載され思考し自身を操作することができる電子通貨(ERC20/ポイント)です。



## 🎯ポイント

1. 指定されたAIが、ユーザーのコメントに応じて通貨全体を操作(生成、配布、剥奪、減少)
2. 通貨を通してAIとユーザーのコミュニケーションを促進
3. SaaSとしてERC20だけでなくポイントなどにも統合可能



## 🐾ユースケース

- キャラクターとファンのコミュニケーションを促進
- 新しいmemeコイン/通貨の生成
- 自動でより多くの人に普及を目指す自己思考型通貨の発行
